# 运行设置（CAD 打开容忍）

- 文件打开等待标准：120 秒
  - 适用范围：通过 `CadController.open_dwg()` 打开的所有 DWG 文件。
  - 理由：部分包含中文/特殊字符或体量较大的 DWG，在实际环境中需要 10 秒以上；统一容忍至 120 秒以提升稳定性。
- 已落实：
  - `cad_automation/controller.py` 中为控制器实例设置 `self.open_timeout_s = 120`；
  - 打开流程内的等待均使用该超时值（空闲等待与“文档加入集合”等待）。
- 使用建议：
  - 命令行传入路径请加引号（尤其包含空格、#、中括号、【】等字符时）。
  - 若仍出现偶发超时，可复现后提供 `artifacts/logs/cad_automation.log` 与具体路径，进一步优化。
