## 使用流程

### 创建新图纸项目

1. **创建图纸**:
   ```bash
   cd D:\claude-tasks
   python project_scripts/create_plot_map.py
   python project_scripts/create_plot_map_2.py
   ```

2. **添加坐标标注**:
   ```bash
   python annotation_scripts/add_leader_labels.py
   ```

3. **整理完成的文件**:
   - 文件会自动保存到 `drawings/YYYY-MM-DD/`

## 注意事项

1. **scripts文件夹**: 不要移动或修改，这是所有脚本的依赖模块
2. **文件占用**: 如果DWG文件正在被CAD使用，需要先关闭CAD
3. **日期文件夹**: 每次创建新图纸时，建议创建当天日期的文件夹
4. **备份文件**: .bak文件是CAD自动备份，可以保留

## 文件说明

### Python脚本分类

**项目脚本** (`project_scripts/`):
- 用于创建新的DWG文件
- 定义图纸的基本几何形状

**标注脚本** (`annotation_scripts/`):
- 用于为已有DWG文件添加标注
- 包含多种标注方式（推荐LEADER命令方式）

**基础模块** (`scripts/`):
- CAD操作的底层函数
- 不应该直接运行，被其他脚本调用
